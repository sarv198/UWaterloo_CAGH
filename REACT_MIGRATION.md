# React Migration - UW CAGH

All Django HTML templates have been converted to React components. The site now runs as a single-page application (SPA) with Django serving the React app and providing a REST API for the blog.

## Project Structure

```
frontend/                 # React app (Vite + React)
├── src/
│   ├── components/       # Layout, Navbar, Footer, MemberCard
│   ├── pages/            # About, Events, Involvement, Team, BlogHome, PostDetail, PostForm, PostDelete
│   ├── data/             # teamData.js (team members)
│   ├── App.jsx
│   ├── main.jsx
│   └── index.css
└── vite.config.js

CAGH/
├── static/frontend/      # Built React assets (generated by npm run build)
├── templates/CAGH/
│   └── react_spa.html    # Django template that loads the React app
├── api_views.py          # REST API for blog posts
├── serializers.py
└── views.py              # Serves React SPA for all frontend routes
```

## Development

### Option 1: Run with Django (production-like)

1. Build the React app:
   ```bash
   cd frontend && npm run build
   ```

2. Start Django:
   ```bash
   python manage.py runserver
   ```

3. Visit http://127.0.0.1:8000/

### Option 2: Run React dev server (hot reload)

1. Start Django (for API):
   ```bash
   python manage.py runserver
   ```

2. Start Vite dev server (proxies API to Django):
   ```bash
   cd frontend && npm run dev
   ```

3. Visit http://localhost:5173/

## Routes

| Path | Component |
|------|-----------|
| `/` | Redirects to `/about` |
| `/about` | About |
| `/events` | Events |
| `/involvement` | Get Involved |
| `/team` | Meet the Team |
| `/blog` | Blog home (posts list) |
| `/blog/post/new` | Create post |
| `/blog/post/:id` | Post detail |
| `/blog/post/:id/edit` | Edit post |
| `/blog/post/:id/delete` | Delete post |

## API

- `GET /api/posts/` - List posts
- `POST /api/posts/` - Create post
- `GET /api/posts/:id/` - Get post
- `PUT /api/posts/:id/` - Update post
- `DELETE /api/posts/:id/` - Delete post

## Static Assets

Team images are served from Django at `/static/images/team/`. The React app references them at that path.
